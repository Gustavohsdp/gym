## :gear: Execute

First clone this repository:

```bash
$ git clone https://github.com/Gustavohsdp/gym
$ cd gym
$ git clone  https://github.com/Gustavohsdp/gym-api
$ cd gym-api
```

### Server

```bash
$ cd gym-api
$ npm i
$ npm run migrate
$ npm run dev
```

### Mobile

```bash
$ cd gym
$ npm i
$ npx expo start
```
